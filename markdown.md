##### Решение задачи:

```python
import math

a = int(input("Введите a: "))
b = int(input("Введите b: "))
c = int(input("Введите c: "))

def S():
  p = (a + b + c) / 2
  return math.sqrt(p * (p - a) * (p - b) * (p - c))

print("Площадь треугольника: ", S())
```
